# Мој дневник - веб апликација за подобрување на менталното здравје

### Проект по предметот ПНУВ

## Опис
„Мој дневник“ е веб апликација која овозможува корисниците редовно да ги бележат своите мисли, чувства и дневни искуства, што помага во препознавање и справување со стрес, анксиозност и емотивни предизвици. Целта е да се поттикне свесност за сопственото ментално здравје како и да им помогне на младите да се мотивираат да градат навики.
